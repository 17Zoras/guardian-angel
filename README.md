# Guardian Angel – Real-Time Emergency Safety Platform

## Overview
A production-grade web application that allows users to trigger emergency alerts,
stream live location data, and notify trusted contacts via SMS.

## Key Features
- Secure Supabase authentication
- Row Level Security (RLS) isolation
- Real-time geolocation streaming
- SMS alerts via Edge Functions
- Progressive Web App (PWA)
- Modular architecture

## Architecture
Frontend: React + Vite + TypeScript  
Backend: Supabase (Postgres + Realtime + Edge Functions)  
Security: RLS policies + server-side secrets  
Deployment: Lovable + Supabase  

## Why This Is Production Ready
- Strict data isolation
- Secure backend messaging
- Realtime subscription handling
- Environment variable validation
- Clean modular structure
