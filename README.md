# Guardian Angel 🛡️🚨

Guardian Angel is a smart safety and emergency assistance platform designed to enhance personal security using modern technology.  
The application provides quick access to emergency services, real-time alerts, and safety-focused features for users during critical situations.

Built with a modern tech stack, Guardian Angel focuses on reliability, fast response systems, and user-friendly interaction.

---

## 🌟 Features

- 🚨 One-tap Emergency SOS
- 📍 Real-time Location Sharing
- 🔔 Instant Emergency Alerts
- 👥 Emergency Contact Support
- 📱 Responsive User Interface
- ⚡ Fast and Secure Backend
- 🛡️ Safety-focused Design
- 🌐 API Integration Support

---

## 🛠️ Tech Stack

- **Frontend:** React / Vite
- **Backend:** Node.js / Express
- **Database:** MongoDB
- **Authentication:** Firebase / JWT
- **Styling:** Tailwind CSS / CSS
- **APIs:** Maps / Notification APIs

## 🚀 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/17Zoras/guardian-angel.git
```

### 2️⃣ Navigate to the Project Directory

```bash
cd guardian-angel
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Setup Environment Variables

Create a `.env` file in the root directory and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
API_KEY=your_api_key
```

---

## ▶️ Run the Project

### Start Development Server

```bash
npm run dev
```

OR

```bash
npm start
```
